Ejercicio 1
En el ejercicio que fue desarrollado en java para aplicar el encapsulamiento, uso de getters y setters, constructores, validacion de datos, metodos estaticos, sobrecarga de metodos y logica condicional. El programa simula un sistema de evaluacion el subsidio en Ecuador y si una persona es elegible.

Se debe ejecutar en el main1 y se solicitara ingresar datos por consola de la siguiente forma:
     
     Scanner sc = new Scanner(System.in);
        System.out.print("Ingrese su nombre completo: ");
        String nombre = sc.nextLine();
        System.out.print("Ingrese su cedula: ");
        String cedula = sc.nextLine();
        System.out.print("Ingrese sus ingresos mensuales: ");
        double ingreso = sc.nextDouble();
        System.out.print("Ingrese la cantidad de vehículos registrados: ");
        int cantidad = sc.nextInt();
        System.out.print("¿Vive en Ecuador? (true/false): ");
        boolean viveEnEcuador = sc.nextBoolean();
